# Data Ownership Matrix

| Data | Owner | Consumers |
| --- | --- | --- |
| Opportunity and blueprint | Control plane | Growth, analytics |
| Product and price versions | Catalog/pricing | Orders, execution |
| Order | Transaction module | Execution, finance, CRM projection |
| Workflow run | Execution plane | Delivery, usage, audit |
| Ledger | Finance module | Settlement, analytics |
| Lead and proof | Growth plane | CRM, quotes |
| Lead/Customer ownership | Channel module | Growth, reseller portal, commission policy |
| Supplier contract, rate, quality | Channel module | Provider routing, finance projection |
| Commission and ProviderPayable | Finance module | Reseller/supplier projections, Settlement |
| Listing and immutable versions | Marketplace governance | Developer marketplace, release review |
| Listing dispute and takedown | Marketplace governance | Security, audit, publication |
| Audit | Governance | Security, compliance |
