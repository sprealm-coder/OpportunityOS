# Context Map

The control plane owns definitions and commercial records. The execution plane consumes immutable product, SKU, workflow, price, and route versions. The growth plane owns leads, proofs, campaigns, and deals. Governance owns permissions, feature gates, risk, and audit. Data and learning consume events and produce advisory scores; they cannot mutate price, ledger, or permissions directly.

Shared identifiers are references, not shared table ownership. Order is the transaction source of truth; external CRM systems are integrations only.
