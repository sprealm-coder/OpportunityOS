# Domain Glossary

- Source and Signal: origin configuration and normalized observations.
- Opportunity and Evidence: a versioned commercial hypothesis and supporting facts.
- BusinessBlueprint: approved, versioned description of a possible business.
- Capability: an abstract ability; ProviderEndpoint is one executable supply option.
- Product and SKU: published promise and purchasable version binding.
- Workflow: controlled execution graph; it never accepts arbitrary uploaded code.
- Usage, ProviderCost, CustomerCharge: separate operational facts.
- LedgerEntry: immutable debit or credit in a balanced transaction.
- OutcomeFeedback: measured result linked back to the source opportunity.
